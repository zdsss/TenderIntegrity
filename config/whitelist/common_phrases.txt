# 医疗采购通用行业惯例表述白名单（用于向量化存储）
# 每行一个独立表述，以#开头的行为注释

# 质保条款
提供不少于一年的质量保证期
自验收合格之日起提供免费维修服务
免费提供备品备件
提供7×24小时技术支持服务
在质保期内提供免费维修和零配件更换
提供不低于两年的整机质保服务
质保期内免费上门维修，响应时间不超过24小时
设备故障时提供备用机或临时替代方案

# 交货条款
货物验收合格后付款
按照合同约定时间交货
货到现场后组织验收
交货期自合同签订之日起计算
货物验收由采购方组织相关专家进行
交货地点为采购方指定地点
运输过程中的风险由供应商承担
货物交付时须附随产品说明书及相关证件

# 售后服务
提供现场安装调试服务
派驻专业工程师进行安装培训
提供操作培训及技术培训
定期进行设备维护保养
提供全程售后技术支持
建立完善的售后服务体系
服务热线保持全年365天畅通
响应时间不超过8小时，到达现场时间不超过24小时
提供年度预防性维护保养服务
维修所需零配件保证供应不少于十年

# 资质要求
具有医疗器械经营许可证
持有效的营业执照
提供相关产品注册证
具备相应的质量管理体系认证
提供产品合格证及检测报告
具备医疗器械生产许可证
通过ISO 13485质量管理体系认证
提供产品在国内合法销售证明文件
具有良好的商业信誉和履约能力

# 投标文件格式
投标文件正本一份副本若干份
投标文件须加盖公章
法定代表人或授权代理人签字
投标保证金按规定缴纳
投标文件密封包装后按时提交
投标截止时间后提交的投标文件不予受理
投标文件须用中文编写
投标有效期为提交截止日后九十天

# 评标标准
综合评分法进行评标
技术标和商务标分开评审
价格分按最低价计算
评标委员会由专家组成
评标结果以评标委员会最终意见为准
技术评分侧重功能实现与技术先进性
商务评分包含售后服务能力与企业信誉

# 合同条款
按照合同约定执行
合同签订后生效
双方协商一致可以变更
合同争议通过协商或仲裁解决
合同受中华人民共和国法律约束
不可抗力导致合同无法履行时双方免责
合同的修改和补充须以书面形式确认
付款方式按合同约定分期支付

# 医疗器械通用声明
产品符合国家相关标准及行业规范
所有设备均为全新正品，非翻新或二手产品
提供原厂授权证明及授权代理商证书
设备软件持续更新，不另行收费
产品具有完整的中文操作界面及说明书
设备通过国家药品监督管理局注册审批

# 采购流程通用表述
本次采购遵循公开、公平、公正原则
采购人有权对投标文件进行质疑和澄清
中标后须按规定时间签署采购合同
严格按照政府采购相关法规执行
投标人须承诺所提供信息真实有效
