# 场景一：标书雷同与语义查重 — 需求设计与规划

## 一、场景定义

在医疗招投标、项目采购、材料申报等场景中，系统对两份或多份长文档进行比对，识别高度雷同、语义改写、模板套用或关键参数重复的问题，并输出可供人工复核的风险报告三个核心场景产品方向说明。

------

## 二、目标

### 2.1 核心目标

- 快速发现可疑雷同内容
- 自动排除正常引用或行业通用表述造成的误报
- 明确展示每处风险及判定理由
- 提供人工复核便利，提高审查效率三个核心场景产品方向说明

### 2.2 用户价值

- **医疗采购审查人员**：减少人工逐页比对工作量
- **招投标监管人员**：快速发现潜在围标或串标行为
- **内审/纪检人员**：高效获得可疑材料线索
- **管理者**：便于内部汇报和复核决策三个核心场景产品方向说明

------

## 三、目标用户

| 用户类型         | 使用场景                       |
| ---------------- | ------------------------------ |
| 医疗采购审查人员 | 处理投标文件、项目申报材料     |
| 招投标监管人员   | 审查投标文件的合规性和雷同风险 |
| 内审与纪检岗位   | 监督标书合规，辅助调查         |
| 管理者           | 审阅风险报告，指导复核决策     |

------

## 四、核心问题与痛点

1. 文档长，人工比对耗时高
2. 普通查重工具仅识别字面重复，无法发现改写或洗稿
3. 法规、标准条文容易导致误报
4. 审核人难快速定位风险片段

**产品定位**：不是“查重工具”，而是“面向监管与审计场景的风险筛查工具”三个核心场景产品方向说明

------

## 五、功能设计

### 5.1 上传与管理

- 支持批量上传标书或申报材料
- 自动解析文档格式（PDF、Word、文本）
- 支持历史标书管理，便于比对

### 5.2 相似度分析

- 文档级、段落级、句子级相似度分析
- 支持语义改写、同义词替换识别
- 可配置敏感字段（价格、关键参数等）

### 5.3 风险标注与报告

- 高风险段落清单
- 风险等级分类（高、中、低）
- 每处风险的判定理由
- 可供人工复核的对照内容

### 5.4 可视化与交互

- 风险摘要页面，突出高风险段落
- 文档对比视图，标出雷同片段
- 可点击展开具体段落内容
- 支持导出风险报告（PDF/Excel）

### 5.5 用户辅助功能

- 风险搜索与筛选（按段落、风险等级）
- 历史记录查询与对比
- 标注和评论功能，支持团队协作

------

## 六、业务流程

1. 用户上传待审标书和参考标书
2. 系统进行文档解析与语义比对
3. 系统生成风险摘要和高风险段落清单
4. 用户查看摘要，点击高风险内容展开详细比对
5. 用户形成复核结论或决定进入人工调查
6. 可导出风险报告进行汇报或存档

------

## 七、产品输出

| 输出类型       | 内容说明                            |
| -------------- | ----------------------------------- |
| 整体风险等级   | 高/中/低或分数化指标                |
| 疑似雷同率     | 文档整体可疑雷同比例                |
| 高风险段落清单 | 每段落编号、风险等级、判定理由      |
| 对照内容       | 可供人工复核的原文对比              |
| 风险报告       | 可导出PDF/Excel，包含摘要和详细内容 |

------

## 八、产品边界

- 不直接认定违法违规
- 不输出最终处罚结论
- 不完全替代人工法务或审计判断
- 主要功能：前置筛查、风险发现、复核辅助三个核心场景产品方向说明

------

## 九、阶段规划

### 9.1 阶段1 — 最小可验证版本 (MVP)

- 支持单份标书与参考标书比对
- 输出整体风险等级和高风险段落清单
- 可导出报告

### 9.2 阶段2 — 功能增强

- 批量比对多份文档
- 支持语义改写识别、敏感字段配置
- 风险等级分类与可视化对比

### 9.3 阶段3 — 平台能力拓展

- 历史文档管理与跨项目比对
- 风险搜索、标注协作、团队复核功能
- 与合同预审和数据风险预警模块打通，形成统一平台

------

## 十、成功指标

1. 平均比对时间减少 >50%
2. 高风险内容命中率 >80%
3. 用户人工复核效率提升 >40%
4. 产品演示获得管理层认可，具备推广价值
